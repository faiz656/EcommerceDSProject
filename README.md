E-Commerce Management System

A console-based C++ e-commerce project demonstrating the use of fundamental data structures such as doubly linked lists, stacks, queues, and binary search trees (BST).

This project simulates a small e-commerce platform with admin and customer panels, along with shopping cart, order processing, and receipt generation.

---

Features

Admin Panel
- Add, update, and delete products
- Add and view categories
- View and update order status
- Secured with a password (`1234`)

Customer Panel
- Browse and view products
- Add products to a cart
- Checkout with receipt generation including name, mobile, address, subtotal, and total
- View recently viewed products (stack)
- View order history

Data Structures Used
- Doubly Linked List → For storing products
- Stack → For recently viewed products
- Queue → For order processing (FIFO)
- BST (Binary Search Tree) → For categories

---

How to Run

1. Open `main.cpp` in any C++ IDE (e.g-Visual Studio).  
2. Compile and run the program.  
3. Choose Admin or Customer panel.  
4. Admin password: `1234`  
5. Use menus to explore all features.
